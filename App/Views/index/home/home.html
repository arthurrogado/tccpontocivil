O que é isso?

Isso se trata do resultado de um Trabalho de Conclusão de Curso de Arthur Rogado Reis, com o tema: DESENVOLVIMENTO DE SISTEMA DE ORÇAMENTO PARA CONSTRUÇÃO CIVIL.

Como usar?
Para usar o sistema, você deve:

1. Navegar o menu Orçamentos.

2. Criar um novo orçamento clicando no botão Novo orçamento.

3. Preencher os dados do orçamento (os obrigatórios são 'nome', 'UF' e 'mês').

4. Clique no botão Visualizar

5. Adicione uma etapa em +Etapa (digite a descrição e salve).

6. Adicione composições ou insumos clicando em +Composição ou +Insumo


A partir daqui você já pode visualizar o relatório em tela, com os somatórios. Mas também tem a possibilidade de:

  • Alterar detalhes do orçamento clicando no botão de editar ()
  • Expandir as etapas clicando sobre.
  • Excluir etapas e itens nas opções que se abrem no
